### General Info
***
NanoCare is a website that offers a platform which make  hospital management and a clinical management more easier and faster.it has both user and admin interface but the admin interface has much more options than ordinary user so it allows you to manage stuffs,patients,payments,expenses,appointment,medical departement and blog.
users can take profit of this platform too:
-read blog articles.
-schedule appointments online;
-check departement availability,see doctors list, create accounts.
